## :floppy_disk: Installation and usage

```
$ git clone https://github.com/dozsolti/wallpaper-app

$ cd wallpaper-app

$ npm install
```
### Run

```
$ npm run start
```


**After that** scan the qr code with your phone using [Expo Go](https://expo.dev/client).


### Linting
```
$ npm run lint
```
